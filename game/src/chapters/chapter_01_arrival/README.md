# Chapter 01: 初到木柵

## 故事大綱

玩家是新搬到木柵的政大研究生，第一次出來探索社區。在校園逛完後，被指引到附近的木柵市場買菜，認識了攤販陳阿姨。陳阿姨第一次見到玩家，會以陌生人的態度親切招呼，並提到附近其他可去的地方。

## 範圍

- **Zones**：`zone_nccu`（政大正門）、`zone_market`（木柵市場）
- **NPCs**：`chen_ayi`（陳阿姨）、`wang_bobo`（王伯伯，可選）
- **任務**：
  - `ch01_first_walk` — 在政大走一圈
  - `ch01_meet_ayi` — 跟陳阿姨對話一次
- **完成條件**：`talked_to_chen_ayi`

## NPC 章節差異

- **陳阿姨**：你不認識玩家，當他是新搬來的學生，親切但保持距離
- **王伯伯**：（可選）路上遇見，會點頭打招呼但不主動搭話

## 進入下一章條件

完成 `ch01_meet_ayi` quest（陳阿姨對話過一次）→ 觸發 flag `chapter_completed_ch01_arrival` → 進入 `ch02_market`
