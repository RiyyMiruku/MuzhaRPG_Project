Pixel Lab API

The API provides endpoints for creating AI-generated pixel art images, rotations, animations, and more. Making it easy for applications to integrate pixel art generation capabilities.

AI Assistant Integration

For use with AI assistants (Claude Code, ChatGPT, Cursor, etc.), use the auto-generated documentation:

https://api.pixellab.ai/v2/llms.txt

Python Client

For convenience, a Python client library is available to simplify integration with your applications. Visit our GitHub repository for installation instructions and examples.

pip install pixellab

Authentication

The API uses a simple token based authentication system. After creating an account, you can find your API token in your account settings. Include this token in all API requests using the Bearer authentication scheme:

curl -X POST https://api.pixellab.ai/v2/create-image-pixflux \
    -H "Authorization: Bearer YOUR_API_TOKEN" \
    -H "Content-Type: application/json" \
    -d '{
        "description": "cute dragon",
        "image_size": {"width": 128, "height": 128}
    }'


Or use the Python client:

import pixellab

client = pixellab.Client(secret="YOUR_API_TOKEN")
client.create_image_pixflux(
    description="cute dragon",
    image_size=dict(width=128, height=128),
)

Image Generation

Create image Pixflux: Generate pixel art from text descriptions.
Image to pixel art: Convert regular images to pixel art style.

Pro:

Generate image (Pro): Generate multiple pixel art images from text (improved quality).
Generate with style (Pro): Generate new images matching the style of reference images.
Generate UI (Pro): Generate pixel art UI elements (buttons, health bars, etc.).
Generate 8 rotations (Pro): Generate 8 directional views of a character or object.
Animation
Animate with skeleton: Generate 4 frames of an animation from skeleton poses.
Animate with text: Generate animation from text descriptions.
Animate with text v2: Generate animation from text (improved quality).

Pro:

Edit animation (Pro): Apply text-based edits consistently across animation frames.
Interpolate (Pro): Generate intermediate frames between two keyframe images.
Transfer outfit (Pro): Transfer outfit/appearance from a reference to animation frames.
Editing
Inpaint: Edit and modify existing pixel art.
Edit image: Edit an existing pixel art image based on a text description.
Resize: Intelligently upscale or downscale pixel art while preserving style.
Rotate: Rotate an object or a character.

Pro:

Edit images (Pro): Edit multiple pixel art images using text or reference images (improved quality).
Inpaint image (Pro): AI-powered inpainting with mask-based editing.
Tilesets & Objects
Create tileset: Create complete top-down tilesets for game development with seamless connections.
Create tileset sidescroller: Create sidescroller platform tilesets for 2D platformer games.
Create isometric tile: Create isometric tiles in various shapes (thick tile, thin tile, block).
Create map object: Generate transparent map objects (trees, items, buildings) for game development.
Character Creation & Management (New)
Create character with 4 directions: Create persistent characters with 4 directional rotations stored for reuse.
Create character with 8 directions: Create persistent characters with 8 directional rotations stored for reuse.
Animate character: Add animations to existing characters with automatic multi-direction processing.
List characters: View all your created characters with metadata and preview images.
Get character details: Access complete character information and all rotation image URLs.
Export character: Download characters as ZIP files with rotations, animations, and keypoint metadata.
Background job status: Monitor the progress of character creation and animation jobs.
